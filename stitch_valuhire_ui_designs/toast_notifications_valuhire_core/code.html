Notification System

A comprehensive showcase of ValuHire's enterprise toast notifications, designed for clarity, actionability, and subtle feedback loops.

design_services System Guidelines

  • check_circle Success: Transient confirmation for completed tasks (e.g., candidate moved, assessment sent). Auto-dismisses.
  • error Error: Persistent until dismissed. Always provides a clear path forward or a retry action.
  • warning Warning: Cautions the user about potential consequences without blocking their immediate workflow.
  • info Info: Neutral updates (e.g., system maintenance notices, background tasks completing).
